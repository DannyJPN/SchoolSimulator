# School Simulator

Pracovni repozitar pro 3D historicky simulator skolni vyuky v ceske vesnicke skole kolem roku 1900.

Aktualne repozitar obsahuje prvni souhrn funkcni specifikace sesbirany z navrhu projektu.

## Obsah

- `docs/specs/00-overview.md` - vysoka uroven projektu a pevne mantinely
- `docs/specs/01-student-campaign.md` - specifikace kampane zaka
- `docs/specs/02-teacher-campaign.md` - specifikace kampane ucitele
- `docs/specs/03-assessment-and-discipline.md` - znamky, kazen, tresty, pochvaly
- `docs/specs/04-school-rules-and-etiquette.md` - skolni rad, etiketa, ritualy
- `docs/specs/05-ui-data-and-visibility.md` - UI, evidence, pohled jednotlivych roli
- `docs/specs/06-open-questions.md` - body, ktere jeste nejsou definitivne uzavrene

Historicky ramec:

- ceska vesnicka skola
- Rakousko-Uhersko
- pevny start kampani v roce `1900`
- jazyk vyuky `cestina`
